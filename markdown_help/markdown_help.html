Markdown帮助文档

本文档借鉴 为知笔记

What

                    Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）。
                    它允许人们「使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML(或者HTML)文档 」

以上是维基百科上Markdown的定义

Why

Markdown 是用最简单的方式，零排版成本，写出布局工整、阅读舒适的笔记。 减少干扰，降低成本，提高效率 ，不管在什么设备上，看起来都是一个工整的样子。

标题

在行首插入1到6个#，分别表示标题1到标题6

            ### 这是标题3
            #### 这是标题4
            ##### 这是标题5

有序列表

        
在行首增加1. 、2. 、3. 
            ,即数字和英文句号，注意句号与文字中间有空格。
            不要求数字之间一定要连续，可以都是1. 

        
            1. 有序列表
            1. 有序列表
            4. 有序列表
        

        
Markdown效果：


        



        
无序列表

        
在行首增加* 或- ，注意与文字之间有空格

        
            * 无序列表1
            * 无序列表2
            - 无序列表1
            - 无序列表2
        

        
Markdown效果：


        



        
插入图片

        
形如：![](url)，注意使用英文符号

        
            ![](trackBloLogo.png)
        

        
        
插入链接

        
形如：[描述](href)，注意使用英文符号

        
            [轨迹博客](博客首页地址)
        

        
Markdown效果：


        



        
粗斜删

        

            粗体：在文字前后添加**
            

            斜体：在文字前后添加*
            

            删除线：在文字前后添加~~
            

            (注意，符号和文字间不要有空格)
        

        
            **粗体**
            *斜体*
            ~~删除线~~
        

        
Markdown效果：


        



        
引用

        
在文字前后添加> ,注意符号与文字间有空格

        
            > 如果你无法简洁地表达你的想法，那只能说明你还不够了解它 ——阿尔伯特·爱因斯坦

            - [x] 已完成事项
